Cruce de Información Excel

Sistema de cruzado de información entre Mercado Pago y Tienda Nube

La función de este software es generar un Excel a partir de los archivos de Tienda Nube y Mercado Pago. En el Excel resultante tendrás la información lista para copiarla y pegarla en el Excel que utilizan para hacer gestión y seguimiento de sus pedidos.

Archivo Excel MERCADO PAGO

Siga las siguientes instrucciones:

1- Ingrese al siguiente link

2- Presione el botón "Crear Reporte", seguido de "Crear"

3- Seleccione el periodo deseado para obtener los cobros realizados (recomendable que coincida con la fecha del archivo de Tienda Nube)

4- Presione "Generar"

Espere unos minutos y descargue el archivo en formato .csv o .xlsx.

Una vez descargado el archivo, cárguelo abajo.

Haz clic para subir o arrastra y suelta

Archivos .csv o .xlsx

Archivo Excel TIENDA NUBE

Siga las siguientes instrucciones:

1- Ingrese a Tienda Nube

2- Diríjase a la sección de Ventas

3- En la parte superior derecha seleccione "Exportar Lista"

4- Presione "Editar filtros"

5- Seleccione un rango de fecha personalizado

6- Ingrese desde y hasta cuando quiere la información requerida

7- Apreta el botón "Filtrar" y luego presione "Exportar Lista"

Una vez descargado el archivo, cárguelo abajo.

Haz clic para subir o arrastra y suelta
